Resource Garbage Management Platform API Documentation

Built with FastAPI, PostgreSQL, Redis, Celery and Docker

Core Features

  • Automatic resource expiration and garbage collection
  • JWT Authentication integration
  • Role-based access control (Admin/User)
  • Audit logging system for resource operations
  • Email verification workflow for users
  • Configurable TTL (Time-To-Live) for resources
  • Background cleanup service with Redis Streams

Resource Management

  • Upload Resources: Files with automatic expiration
  • Auto-Cleanup: Redis-powered garbage collection
  • Expiry Notifications: Email alerts before resource deletion
  • TTL Override: Admins can modify expiration times

User Management

  • Register User: Create account with email verification
  • Login: JWT token generation
  • Role Management: Admin privileges for resource control

Admin Features

  • Audit Logs: Track all resource operations
  • Resource Management: List/delete expired resources
  • TTL Configuration: Set custom expiration times
